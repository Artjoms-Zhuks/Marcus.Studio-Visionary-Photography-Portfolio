# Marcus.Studio | Visionary Photography Portfolio

A premium, minimalist photography portfolio prototype built with **React** and **Vite**. This project focuses on high-impact visual storytelling and a seamless user experience.

## ✨ Key Features

- **Minimalist Design:** Clean "Visionary" aesthetic tailored for creative professionals.
- **Smooth Navigation:** Custom implementation of smooth scrolling between sections for a refined feel.
- **Responsive Layout:** Fully optimized for all screen sizes, from mobile phones to large monitors.
- **Performance:** Optimized assets and fast loading times thanks to the Vite build system.

## 🛠️ Tech Stack

- **React 18** - UI Library
- **Vite** - Next-generation frontend tooling
- **CSS3** - Custom styling with responsive media queries

## 📝 Project Status: Prototype

This repository is a **technical prototype**.

- Images and videos are placeholders used to demonstrate the visual direction.
- The project showcases frontend architecture, component structure, and responsive design skills.

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone [your-repository-link]
   ```

Install dependencies:
---- npm install

Run the development server:
---- npm run dev

Build for production:
---- npm run build
